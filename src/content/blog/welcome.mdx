My personal portfolio, built to showcase the projects I've been working on, the technologies I enjoy, and the things I'm learning along the way.

## Stack

- Next.js
- React
- Tailwind CSS v4
- TypeScript
- shadcn/ui
- Motion

## Features

- Minimal, responsive design
- Light & Dark mode
- MDX-powered blog and documentation
- Dynamic Open Graph images
- SEO optimized (JSON-LD, sitemap & robots)
- AI-friendly with `llms.txt`
- PWA support
- Component registry powered by shadcn

## Projects

This portfolio also serves as home to my open-source work, including:

- **Shover** — A modern shadcn/ui registry for beautifully crafted components, motion effects, reusable blocks, and templates.
- Personal projects, experiments, and developer tools.
